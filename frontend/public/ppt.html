API使用示例

API EXAMPLE - 完全兼容OpenAI SDK，无缝迁移

Python SDK使用

仅需修改两行配置，即可完成从OpenAI到ThreeRouter的切换。

📦

安装SDK

pip install openai

🔧

配置客户端

base_url & API Key

🚀

调用模型

completions/chat completions

⚡

智能路由

自动选择最优模型

代码示例： 
from openai import OpenAI

client = OpenAI(
    api_key="your_threerouter_key",
    base_url="https://api.threerouter.com/v1"
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role": "user", "content": "Hello"}]
)
