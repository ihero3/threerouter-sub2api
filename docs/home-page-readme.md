## 7. 常见问题解答（FAQ）

### 7.1 关于服务

**Q：你们是传统代理服务吗？**

A：不是。我们不是传统代理服务，而是企业级 AI API 网关。我们的上游服务来自 AWS、GCP、Azure 等官方顶级云厂商；对于开源模型，也直接来自模型厂商或全球认可的推理服务，确保链路合法、透明、可审计。

**Q：你们的模型资源来自哪里？**

A：我们的资源来自官方云厂商、AI 平台、模型厂商以及全球认可的推理服务，帮助确保访问稳定、采购合规、交付链路可溯源。

**Q：你们会存储用户请求数据吗？**

A：不会。网关以安全透传和路由为核心设计，不会将客户提示词或响应内容用于模型训练，并尽量减少运营安全所需之外的数据留存。

**Q：支持哪些模型？**

A：我们覆盖主流前沿模型和高性价比模型，包括 DeepSeek、Qwen、Kimi、MiniMax、GLM 以及 Seedance 等部分视频生成模型。

### 7.2 价格与计费

**Q：这是包月订阅吗？**

A：不是。额度会按实际模型用量扣减，您只为真实消耗付费，不需要为未使用的固定月费买单。

**Q：用量如何计费？**

A：文本模型按 token 用量计费；图片、视频、语音和其他多模态模型则根据对应供应商的计费单位扣费，例如图片张数、视频秒数或音频时长。

**Q：价格是否透明？**

A：透明。价格会按模型和计费单位展示，您可以在使用前预估成本，并在账号后台查看实际消耗。

**Q：团队可以共享额度吗？**

A：可以。团队和企业可通过共享额度池、权限控制和用量分析统一管理，让支出更可控、更可预测。

### 7.3 集成与使用

**Q：是否兼容 OpenAI API？**

A：兼容。网关完全兼容 OpenAI API 格式，大多数情况下只需要替换 base URL 和 API key 即可开始调用模型。

**Q：迁移成本高吗？**

A：对于已有 OpenAI 兼容客户端的项目，通常只需修改一行配置。SDK、Agent 和工作流工具可以继续使用熟悉的请求格式。

**Q：支持故障转移和智能路由吗？**

A：支持。我们通过多供应商路由、负载均衡和故障转移策略提升可用性，减少单一上游限制带来的中断。

**Q：可以用于生产环境吗？**

A：可以。网关面向生产场景设计，提供标准化 API、高可用路由、用量可观测能力以及面向企业的合规实践。
